Installation Guide
========

Download
------
Simply download this "package_" to your workspace and build it.


.. code-block:: none

    cd your_ws

    colcon build


Requirements
------
The requirements for running this project are:
  - NetworkX_
  - numpy_
  - cbs-mapf_
  - pygame_
  - tkinter
